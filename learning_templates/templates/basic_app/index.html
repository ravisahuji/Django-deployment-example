  • To create our own template filter :- we have to follow some convention...
    1. Create a folder (templatetags) inside 'app' folder... at the label of models.py file
    2. create a file (__init__.py) inside 'templatetags' folder
    3. create a file (with any name eg: my_extras.py)

Now cutting the original text "{{text}}" just the below of this paragraph by using 'default filter' => "cut"

{{ text|cut:'hello' }}


Now, original text "{{text}}" into upper case just the below of this paragraph by using 'custom filter' => "upCase"

{{ text|upCase }}


Now cutting the original text "{{text}}" just the below of this paragraph by using 'custom filter' => "text_replace"

{{ text|text_replace:'hello' }}
